# Portfolio

This portfolio is deployed using GitHub Pages. Visit the live site at:

- Responsive design
- Custom cursor effects
- Particle animation canvas
- Smooth scrolling

- Interactive project cards with 3D tilt effects

## 📝 Technologies

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Google Fonts (Orbitron, DM Mono, Syne)
