# Portfolio

Personal portfolio website for Rahil M — MBA candidate in Operations, Supply Chain, and Business Analytics.

## 🚀 GitHub Pages

This portfolio is deployed using GitHub Pages. Visit the live site at:
`https://rvim25mba033rvim-blip.github.io/Portfolio/`

## 📁 Structure

- `index.html` - Main portfolio page with embedded styles and scripts

## 🛠️ Features

- Responsive design
- Custom cursor effects with touch support
- Particle animation canvas
- **Enhanced scrolling features:**
  - Smooth scrolling with navbar offset
  - Scroll progress indicator at the top
  - Scroll-to-top button
  - Parallax scrolling effects
  - Scroll-triggered reveal animations
  - Active navigation link highlighting
- Modern dark theme with accent colors
- Interactive project cards with 3D tilt effects

## 📝 Technologies

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Google Fonts (Orbitron, DM Mono, Syne)
